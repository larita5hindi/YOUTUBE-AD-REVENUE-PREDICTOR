                                                                                                       YouTube Ad Revenue Predictor

Project Overview

The YouTube Ad Revenue Predictor is a Machine Learning project developed using Python and Streamlit. The application predicts the estimated advertising revenue of a YouTube video based on user-provided performance metrics such as views, likes, comments, watch time, video length, subscribers, category, country, and device.

The objective of this project is to demonstrate the complete Machine Learning pipeline, from data preprocessing and model training to deployment through an interactive web application.

---

Problem Statement

Estimating YouTube ad revenue manually is difficult because it depends on multiple factors. This project uses historical data and a Linear Regression model to predict the expected ad revenue based on video performance metrics.

---

Technologies Used

---

Machine Learning Pipeline

1. Data Collection

- Load the YouTube ad revenue dataset.
- Inspect the dataset structure and features.

2. Data Preprocessing

- Handle missing values.
- Remove unnecessary columns.
- Prepare input and target variables.
- Encode categorical features where required.

3. Feature Selection

Input Features:

- Views
- Likes
- Comments
- Watch Time
- Video Length
- Subscribers
- Category
- Country
- Device
- Engagement Rate

Target Variable:

- Ad Revenue (USD)

4. Train-Test Split

- Split the dataset into training and testing sets using "train_test_split()".

5. Model Training

- Train a Linear Regression model using Scikit-learn.

6. Model Evaluation

- Evaluate the model using R² Score and prediction results.

7. Model Saving

- Save the trained model using Joblib as:
  "youtube_revenue_model.pkl"

8. Deployment

- Build an interactive Streamlit application.
- Accept user inputs.
- Load the saved model.
- Predict and display estimated ad revenue instantly.
